# Virus Propagation Simulation

This project is a Python-based simulation designed to model virus propagation across different age groups using scenario-based analysis.

The program analyzes populations, selects the appropriate infection scenario based on age ranges, and generates statistical results and visualizations.

## Project Features

- Scenario-based virus propagation model
- Age group classification
- Population analysis and statistics
- Decision tree visualization
- Graphical data representation
- Interactive simulation for custom inputs

## Technologies Used

- Python
- Jupyter Notebook
- Matplotlib
- Data structures and algorithms

## Project Structure

virus_propagation.ipynb  Main notebook containing the simulation and analysis

## How It Works

The simulation works by defining several virus propagation scenarios based on age groups.

Each scenario contains parameters that determine how the virus spreads within a population.

The program:

1. identifies the correct scenario based on age
2. simulates virus spread for a population
3. analyzes results
4. generates visualizations and statistics

## Visual Components

The project includes graphical representations such as:

- decision tree diagrams
- population analysis charts
- statistical visualizations

These help better understand the propagation patterns.
